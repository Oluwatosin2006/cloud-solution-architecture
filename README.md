# Enterprise Cloud Architecture with Microservices & Kubernetes

## Overview
This project demonstrates a scalable cloud architecture designed for high availability, microservices deployment, and automated CI/CD pipelines

## Architecture Highlights
- Microservices deployed using Kubernetes
- CI/CD pipeline using Microsoft DevOps
- Highly available cloud infrastructure
- Secure networking and load balancing
